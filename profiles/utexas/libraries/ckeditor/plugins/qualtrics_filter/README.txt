Installation:

Do the following steps to add qualtrics_filter button to the CKEditor toolbar:

   1. Enable the utexas_qualtrics_filter module.

   2. Move the qualtrics_filter folder to your libraries folder (it should be there already if you are using the latest
   version of the UTDK).
